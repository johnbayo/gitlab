# gitlab
gitlab deployment in kubernetes<br/>
<br/>
Requirements<br/>
Kubernetes cluster<br/>
Kubectl installed and configured on control machine<br/>
<br/>
### usage
<br/>
<pre>
kubectl apply -f ./
</pre>
<br/>
